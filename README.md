##ECAFRIO - Projeto Acadêmico
Website institucional desenvolvido para a disciplina de Desenvolvimento Web, utilizando HTML5, CSS (W3.CSS) e JavaScript puro.

##🎯 Objetivo
Demonstrar aplicação de:

Estrutura semântica HTML5.
Layout responsivo (Desktop vs. Mobile).
Manipulação de DOM para interatividade (Menu Sidebar).



##📂 Estrutura
/
├── index.html      (Código principal)
└── IMG/            (Pasta com logos: logo.jpg, lg-logo.png, etc.)



##🚀 Como Rodar
Salve o código como index.html.
Crie a pasta IMG e adicione as imagens necessárias.
Abra o arquivo no navegador.



##⚙️ Tecnologias
HTML5 & W3.CSS (Estilização e Grid)
Font Awesome (Ícones)
JavaScript (Controle do Menu)
